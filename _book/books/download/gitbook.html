gitbook

gitbook安装

链接：https://www.jianshu.com/p/421cc442f06c

1.安装gitbook

sudo npm install gitbook-cli -g

2.查看版本，确认安装成功

gitbook -V 或者 gitbook ls（查看安装的所有版本）

3.初始文件

gitbook init

4.运行服务

gitbook serve

5.生成一个 _book 文件夹

gitbook build

6.安装插件

gitbook install
