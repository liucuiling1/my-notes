#### git使用

链接：https://www.cnblogs.com/Jake-Chen-World/p/7991797.html

1.克隆项目：
> git clone ‘链接地址'

2.查看分支：
> git branch

3.切换分支：
> git checkout ‘分支的名字’

4.react 启动（可查看md文件）
>npm run start/div/server

5.提交项目
> git add .  
> git commit -m '文字描述'  
> git push   

6.看文件更新状态
> git status

7.下载：
> git pull

```
注：如果有代码冲突时要把代码下载下来，修改整理代码，再提交
```
